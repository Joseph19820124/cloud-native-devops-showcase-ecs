# ECS task definitions

In this project the task definitions are generated by Terraform
(`terraform/modules/ecs/main.tf`) rather than kept as standalone JSON — that
keeps image tags, log groups, IAM roles and Service Connect wiring in one place.

This directory is the ECS analogue of the EKS edition's `k8s/` folder. If you
prefer to drive deploys with `aws ecs register-task-definition` /
`aws ecs update-service` (or the `amazon-ecs-deploy-task-definition` GitHub
Action) instead of Terraform, drop the rendered JSON here. You can export the
Terraform-managed definitions any time with:

```bash
aws ecs describe-task-definition \
  --task-definition showcase-dev-backend \
  --query taskDefinition > ecs/task-def-backend.json
```
